# Adil Kaji Data Analyst Masterpiece: E-commerce Growth Analysis

## Project Overview
This project demonstrates a full end-to-end data analysis lifecycle focused on E-commerce conversion funnels. The goal is to identify where customers drop off in the purchasing process and provide actionable recommendations to increase revenue.

## Tech Stack
* **Language:** Python 3.14 (Pandas, NumPy)
* **Database:** SQL
* **Environment:** GitHub & Jupyter Notebooks

## The Data Journey
1. **Data Acquisition:** Generated a synthetic dataset of 10,000 user sessions with realistic e-commerce behaviors.
2. **Data Wrangling:** Performed rigorous data pre-processing to handle missing values and remove inconsistencies.
3. **Exploratory Analysis:** Conducted a "Deep Dive" into user events to calculate conversion rates.
4. **Insight Delivery:** Identified patterns and trends to build a data-driven business strategy.
